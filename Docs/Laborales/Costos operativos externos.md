# 11\. Modelo de Cobro

Selecciona cómo cobrarás.

* Mensualidad

* Implementación \+ suscripción

* Licencia

* Mantenimiento anual

## Estructura

* Pago inicial: $26 000 MXN

* Mensualidad (20 meses): $3 000 MXN

* Tiempo de desarrollo: 6 semanas

## Precio Final

**Precio total sugerido:** $86 000 MXN

Propuesta del plan de pagos:

El pago total del sistema será de $86 000 MXN, 

Planeados en un pago inicial del 30% ($26 000 MXN) por el desarrollo completo del sistema.

Luego se dividiría a mensualidades de $3000 MXN hasta completar el pago total. El sistema tendrá una garantía de aquí hasta que se termine de pagar el sistema. Esto quiere decir que, por cualquier corrección, parche o actualización necesaria para que el sistema funcione correctamente, nosotros nos encargamos de solucionar dichos asuntos técnicos.

Una vez se termine de pagar el sistema, si se le quiere dar mantenimiento adicional, se hará una cotización aparte.

(Nota: aclararle a Marcelo que estamos proponiendo este plan de pagos porque es lo que se nos acomoda a nosotros, pero el cómo se maneje el pago es flexible, para que pueda, por ejemplo: pagar más de inicial o más por mes y así acabar de liquidar en menos tiempo.)

Si se desea que se implementen nuevas funcionalidades, se realizará un cobro adicional.

El tiempo estimado de entrega del sistema es de 6 semanas, con entregas parciales sobre el progreso del desarrollo cada 15 días. Al alcanzar la liquidación del pago total del sistema, se le entregará el sistema por completo, siendo ahora suyo y concediéndoles el código de este para futuras decisiones que tomen con él.

Nosotros ofrecemos asesoría para las siguientes implementaciones, sin embargo, los costos de servicios operativos (hosting, dominio web, etc.) no están incluidos en el plan de trabajo y correrían por parte del cliente.
